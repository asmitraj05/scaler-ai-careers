🔗 SMART "CONNECT ON LINKEDIN" FEATURE

When user clicks "🔗 Connect on LinkedIn" button:
  1. Opens company's LinkedIn page
  2. Navigates to People section
  3. Auto-searches for "HR"
  4. Shows HR professionals at that company
  5. User can browse and connect manually

================================================================================
                           URL GENERATION LOGIC
================================================================================

FORMAT: https://www.linkedin.com/company/{slug}/people/?keywords=HR

EXAMPLES:
  • Razorpay          → https://www.linkedin.com/company/razorpay/people/?keywords=HR
  • Google            → https://www.linkedin.com/company/google/people/?keywords=HR
  • Goldman Sachs     → https://www.linkedin.com/company/goldman-sachs/people/?keywords=HR
  • Microsoft Inc.    → https://www.linkedin.com/company/microsoft/people/?keywords=HR

FEATURES:
  ✅ Automatic company name cleaning (removes Inc., LLC, Ltd., etc.)
  ✅ Space-to-hyphen conversion (multi-word companies)
  ✅ Lowercase conversion for LinkedIn slug format
  ✅ Special character handling
  ✅ URL-safe encoding
